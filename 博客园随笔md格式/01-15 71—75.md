# 71、请实现一个装饰器,通过一次调用使函数重复执行5次。

```python
# 答案
def again_func(func):

    def inner(*args, **kwargs):

        for line in range(5):

            func(*args, **kwargs)

    return inner

@again_func
def func1():
    print('from func1...')

func1()
```
# 72、python 一行 print 出 1~100 偶数的列表, (列表推导式, filter 均可)
```python
# 答案
print([i for i in range(1, 101) if i % 2 == 0])

print(list(filter(lambda x: x % 2 == 0, range(1, 101))))
```
